-- For bindless resources: --

Editing project settings only changes the shader model for FUTURE hlsl files. IT DOES NOT CHANGE THE SHADER MODELS OF EXISTING FILES!
Thus, need to set the shader model to 6.6 for each existing shader manually!!!

When the shaders each have a different model assigned to them, the pipeline state will appear null, coupled with this error under "Output":
D3D12 ERROR: ID3D12Device::CreateGraphicsPipelineState: Graphics pipeline state mixes shader model 6.0 or above with 5.1 or below. [ STATE_CREATION ERROR #1046: CREATEGRAPHICSPIPELINESTATE_SHADER_MODEL_MISMATCH]
